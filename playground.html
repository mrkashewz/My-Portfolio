Playground

Altimetrik's Software Learning App

Introduction

Summary

Playground is an online learning portal which will allow companies to train their employees on the latest software programs and languages such as Javascript, Adobe Creative Suite and Angular. In this app, users are first given preliminary tests to assess their knowledge. If they pass, they are then assigned a mentor who will assist them as they progress in the course. Playground was created by Altimetrik, a technology solutions company with locations worldwide. The app is projected to launch in December of 2017.

Problem

When working with large teams across the world, miscommunications and problems eventually occur. One example of this was when I was working with the back-end team on implementing an API structure on my HTML pages. The API should have been sent to me in a JSON format but instead was sent with HTML formatting which didn’t display as intended.

Solution

After researching and collaborating with other colleagues I found a way to display the API given to me using the InnerHTML property. This solution not only solved my goal of implementing the API but also fixed the error the back end-team made which was useful since we were coming close to meeting a deadline.
